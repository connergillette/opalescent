Beautiful, colorful, eye-catching CSS buttons.

Opalescent offers styles for animated gradient buttons in a variety of colors and styles using vanilla CSS. They are easily customizable to reflect your brand's colors, motion curves, and shapes.

Getting started

After downloading the CSS file using the button at the top of the page, move or copy it into your project directory. Then, link the file to your HTML page like so:

<link href="./yourfilepath/opalescent.css" rel="stylesheet" type="text/css">

To create a basic Opalescent button, follow the following formats to produce the buttons above:

<div class="opal">
  <div class="opal-solid">
    Solid
  </div>
</div> <div class="opal">
  <div class="opal-soften">
    Softened
  </div>
</div> <div class="opal">
  <div class="opal-framed">
    Framed
  </div>
</div>

The inner classes are added to the inner element to dictate the general look of the button.

The style classes are added to the outer element to dictate the color gradient of the button.

Modifiers are also added to the outer element with a style class (see below) to change functionality of the button.
