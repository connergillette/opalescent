Beautiful, colorful, eye-catching CSS buttons.

Opalescent offers styles for animated gradient buttons in a variety of colors and styles using vanilla CSS. They are easily customizable to reflect your brand's colors, motion curves, and shapes. Use them sparingly to make high-importance actions pop.

Getting started

After downloading the CSS file using the button at the top of the page, move or copy it into your project directory. Then, link the file to your HTML page like so:

<link href="./yourfilepath/opalescent.css" rel="stylesheet" type="text/css">

To create a basic Opalescent button, use the following formats to produce the buttons above:

<div class="opal">
  <div class="opal-solid">
    Solid
  </div>
</div> <div class="opal">
  <div class="opal-soften">
    Softened
  </div>
</div> <div class="opal">
  <div class="opal-framed">
    Framed
  </div>
</div>

The inner classes are added to the inner element to dictate the general look of the button.

The style classes are added to the outer element to dictate the color gradient of the button.

Modifiers are also added to the outer element with a style class (see below) to change functionality of the button.

Customizing color gradients

Opalescent uses the linear-gradient property for generating the button backgrounds. Here is an example of the color gradient for the default .opal color.

linear-gradient(135deg, #ff0000b0 19%, #0000ff80 80%, #00a2ff50 90%) no-repeat;

To apply this color to a button, add the linear-gradient value above as the background of the :before pseudo-element of your new class, like so:

.opal-newcolor:before {
  linear-gradient(135deg, #ff0000b0 19%, #0000ff80 80%, #00a2ff50 90%) no-repeat;
}

Then, add your new class as a style class to your outer element, like so:

<div class="opal opal-newcolor">
  <div class="opal-soften">
    New Color
  </div>
</div>

Customizing gradient animation

These buttons use a basic transform: rotate(360deg) animation running at 5 seconds at default and 2 seconds on hover. To introduce some variation, this animation is partnered with another animation that moves the background back and forth at a fixed interval (8 seconds). These animations are applied to the :before pseudo-element along with the background property.

Customizing shapes

With some customization, the animated gradient effect can be applied to a container of any shape and size, as demonstrated by the header of this site. The main properties that usually need tweaking include the :before pseudo-element (specifically, the size and position) and the degree of the sway animation. I've included the .opal-header class featured in this site's as a starting point for creating larger Opalescent elements, but your mileage may vary.

Luckily, since this project uses stock CSS, it can be easily tweaked to fit the needs of any project. Enjoy!
